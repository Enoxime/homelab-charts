
# Collection of helm charts for my homelab

![Release Charts](https://github.com/Enoxime/homelab-charts/actions/workflows/release.yaml/badge.svg?branch=main)

## Usage

[Helm](https://helm.sh) must be installed to use the charts.
Please refer to Helm's [documentation](https://helm.sh/docs/) to get started.

Once Helm is set up properly, add the repository as follows:

```console
helm repo add homelab https://enoxime.github.io/homelab-charts
```
